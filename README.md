# 🍽️ Restaurant Management System

## 📖 Overview

The Restaurant Management System is a Python-based application developed to simplify restaurant operations. It enables efficient order handling, bill generation, and menu management through a simple console-based interface.

---

- Browse the restaurant menu
- Place customer orders
- Calculate the total bill with service tax
- Generate an itemized receipt
- Save billing information
- Use the integrated calculator
- Start a new order whenever required

## ✨ Features

- 🍴 Menu Display
- 🛒 Order Management
- 💰 Automatic Bill Calculation
- 🧾 Receipt Generation
- ⚡ Easy-to-use Interface
- 📋 Organized Workflow

---

## 🛠️ Technologies Used

- Python 3.x
- Time module for date and bill numbers
- Tkinter for GUI
- Filedialog and messagebox for file interactions and alerts

## 🚀 How to Run

1. ## Installation

Follow these steps to set up and run the project on your local system:

### 1. Clone the Repository

```bash
git clone https://github.com/pariarora2526/Restaurant-Management-System.git
```

### 2. Navigate to the Project Directory

```bash
cd Restaurant-Management-System
```

### 3. Ensure Python is Installed

Make sure Python 3.x is installed on your system.

### 4. Run the Application

```bash
python class12_restaurant.py
```

The Restaurant Management System will start, and you can begin placing orders through the console interface.
